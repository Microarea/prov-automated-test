# QA Automation – Playwright

Repository unico per **test API** e **test UI** basati interamente su **Playwright**.

## Struttura
- `api-tests/` → test automatici dei microservizi
- `ui-tests/`  → test end‑to‑end frontend

## Requisiti
- Node.js 18+ (consigliato 20)
- VS Code

## Avvio rapido
```bash
npm install
```

### API
```bash
cd api-tests
npm test
```

### UI
```bash
cd ui-tests
npm test
```
